# WA/VEウェブサイト

## 概要

pdfデータを参考に、Webサイトを作成する。
Wordpress + Next.jsでヘッドレスCMSを利用する。
Wordpressの経験が少ないので、Wordpress側の設定ドキュメントが欲しい。
サーバーが無いので、ローカルでデバッグ、設定できる仕組みがあれば教えて欲しい。

pdfを読めない場合は教えて欲しい。
pdfから、SVG要素（ボタンや、背景など）必要なものをまとめて教えて欲しい。

## コンテンツ管理

### WORKS

- ハッシュタグリスト

- WAVEの役割
- 参加アーティスト
- URL
- 企業の役割
- ギャラリー画像
- Youtube埋め込み
- クレジット
